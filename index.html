SoftCRM 4.0 고도화 전략

현재 SoftCRM 3.0, 3.5의 문제점, 부족한 사례와 내용을 찾아내고 개선할 방안에 대해 공유하고 토론합니다.

ZTNA 기반 보안 강화

경계가 사라진 하이브리드 업무 환경에서 기존 VPN의 한계를 극복하고, 사용자 중심의 강력한 보안 체계를 수립하는 방안을 제시합니다.

Cloud Native 인프라 전환

온프레미스(사내 구축형) 환경에서 클라우드로의 안정적인 마이그레이션 전략과 이를 뒷받침하는 네트워크 아키텍처 설계를 다룹니다.

AIOps를 통한 운영 자동화

네트워크 장애를 사전에 예측하고 자동으로 대응하여 운영 효율성을 극대화하는 인공지능 기반 운영 솔루션을 소개합니다.
